# Bash and Linux Basic Commands

- Where am I
```
pwd
```

- How to create a Dir
```
mkdir name_of_file
```

- How to create a file
```
touch name_of_file
```

- How to change dir
```
cd dir_name
```

- How to come out of dir
```
cd ..
```

- Who am I
```
uname
```

- List dir and all dir
```ls``` or ```ls-a```
